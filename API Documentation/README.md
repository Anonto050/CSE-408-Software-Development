# CineConnect User API Documentation

## Overview
API for login/registration, user profile, and landing page display.

## Base URL
`https://localhost:3000`

## Endpoints

### User Login
| Method | Endpoint      | Summary          | Description                                   | Request Body                                                                                                                                                                                                                                                                             | Responses                                                                                                                                                                                                                               |
|--------|---------------|------------------|-----------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| POST   | /v1/login     | User Login       | Authenticates user and returns a token.       | <pre>{ <br>  "username": "string",<br>  "password": "string"<br>}</pre>                                                                                                                                                                                                                   | **200**: <br> <pre>{ <br>  "token": "string", <br>  "user": {<br>    ...<br>  }<br> }</pre><br> **401**: <br> <pre>{ <br>  "message": "string"<br> }</pre>                                                                                 |

### User Logout
| Method | Endpoint      | Summary          | Description                                   | Request Body | Responses                                                                                                                                                                                                                               |
